---

### ¿Qué es el *Score de Disciplina*?
Es el número (0–100) que resume tu constancia combinando **asistencia al gym + rutina
completada + nutrición**. Es el corazón de CoachPro: tú y tu coach lo ven de un vistazo.

- 🟢 **75–100** — alto (verde lima)
- ⚪ **50–74** — medio
- 🟠 **menos de 50** — bajo (naranja), toca apretar
